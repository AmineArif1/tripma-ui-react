## Run Locally

#### Clone this repository
```bash
  git clone https://github.com/surajboniwal/tripma-ui-react
```

#### Change the directory
```bash
  cd tripma-ui-react
```

#### Install the required dependencies
```bash
  npm install
```
#### Start the development server
```bash
  npm run start
```
